# SHUTER MODERATION BOT

A Discord.js v14 TypeScript moderation bot with slash commands for banning, kicking, timeouts, warnings, message purging, and mod action logging.

## Commands
- `/ban` `/unban` `/kick` `/timeout` `/warn` `/warnings` `/clearwarns` `/purge` `/ping` `/userinfo`

## Setup
1. Add `DISCORD_TOKEN` and `CLIENT_ID` secrets
2. Run `pnpm install`
3. Run `pnpm run deploy` to register slash commands
4. Run `pnpm run dev` to start the bot
